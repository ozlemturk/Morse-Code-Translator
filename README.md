# Morse Code Translator (Tkinter GUI)

A simple Python application that converts English text into Morse code and converts Morse code back into readable text.  
The project is built using Python and Tkinter for the graphical interface.

---

## Features

- Translate text into Morse code
- Decode Morse code back into English text
- User-friendly GUI built with Tkinter
- Supports letters, numbers, and basic punctuation marks

---

## Supported Characters

### Letters
- A–Z

### Numbers
- 0–9

### Punctuation
- `.` `,` `?` `/` `@`

### Rules
- Spaces in text are converted into `/`
- Unknown characters are ignored or displayed as `?`
- Words in Morse code must be separated with `/`
